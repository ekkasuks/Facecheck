1
ไปที่ Google Cloud Console → สร้างโปรเจค
2
APIs & Services → Credentials → Create OAuth Client ID → Web application
3
Authorized JavaScript origins: https://username.github.io
4
Authorized redirect URIs: .../oauth2callback.html
5
แก้ scripts/config.js → GOOGLE_CLIENT_ID
⚠️ เกิดข้อผิดพลาด
