⚙️ ยังไม่ได้ตั้งค่า Google OAuth — ปุ่ม "Sign in with Google" ยังไม่ทำงาน
1
ไปที่ Google Cloud Console → สร้างโปรเจค
2
APIs & Services → Credentials → Create OAuth 2.0 Client ID → เลือก "Web application"
3
Authorized JavaScript origins: ใส่ URL เว็บของคุณ เช่น https://username.github.io
4
คัดลอก Client ID (รูปแบบ xxxx.apps.googleusercontent.com)
5
เปิดไฟล์ scripts/config.js แก้ไข GOOGLE_CLIENT_ID: 'วาง Client ID ที่นี่'
⚠️ เกิดข้อผิดพลาด
